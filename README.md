# 💼 MSME AI Business Intelligence System

AI-powered platform to analyze business opportunities using Machine Learning and real MSME data.

## Features
- Profit prediction
- Success probability estimation
- Risk analysis
- Real MSME data integration
- AI business consultant

## Tech Stack
- Python, Scikit-learn
- FastAPI
- Streamlit
- Groq LLM

## Run Project

Backend:
uvicorn backend.main:app --reload

Frontend:
streamlit run frontend/Home.py
